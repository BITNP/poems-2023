# 计算器

这是一个计算器，能实现以下功能
1. 四则运算、乘方、开方
2. sin，cos，tan，ln计算
3. 可直接输入e和pi，计算器将自动替换为2.718281和3.141592
4. 可以检查算式是否正确
5. 可以检查算式是否无意义(如除以零，零乘方零)
6. 可通过括号改变运算优先级
7. 括号内采用运算符优先级，即先乘除后加减

运行环境：windows

|--------------------------------------------------|-------------|

| (((2.3+4.6*3-(2^  2))+lne/cos4.234/2))           | 13.1        |
| 4-3*2                                           | -2          |
| 2(5)                                            | 请输入正确的算式 |
| (3*(0^0)-6)+7                                   | 算式无意义   |
